# OrisSemesterWork1
Семестровая работа 1 представляет из себя реализацию базового http сервера на базе httplistener, собственного шаблонизатора и ORM системы. (БД - PostgreSQL)
Для того чтобы протестировать работу сервера самостоятельно необходимо:
1)Открыть СУБД
2)Выполнить SQL запросы из файла Запросы.docx(написаны для PostgreSql)
3)В работе используются абсолютные пути, так что скорее всего придется закинуть проект в папку Пользователи(Users)\source\repos, проверить имя тома(диска), а также поменять имя user-а. Пример:@"C:\Users\yar10\source\repos\MiniHttpServer\MiniHttpServer\Template\Page\hotels.thtml"
4)Открыть файл с решением
5)Запустить программу
6)Ввести в поисковую строку адрес указанный в консоли

Дополнительно:
1)Для проверки работы admin-панели потребуется выдать user-у роль "admin". Сделать это можно путем создания нового user-a или update-ом роли у старого. Пока user не admin - админ панель не доступна.
