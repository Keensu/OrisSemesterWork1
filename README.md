# OrisSemesterWork1
Семестровая работа 1 представляет из себя реализацию базового http сервера на базе httplistener, собственного шаблонизатора и ORM системы. (БД - PostgreSQL)
Для того чтобы протестировать работу сервера самостоятельно необходимо:
1)Открыть СУБД
2)Выполнить SQL запросы из файла Запросы.docx(написаны для PostgreSql)
3)Открыть файл с решением
4)Запустить программу
5)Ввести в браузере адрес указанный в консоли

Дополнительно:
Для проверки работы admin-панели потребуется выдать user-у роль "admin". Сделать это можно путем создания нового user-a или update-ом роли у старого. Пока user не admin - админ панель не доступна.
